# FormUp API

Backend service untuk platform FormUp yang menyediakan REST API untuk aplikasi Web dan Mobile.

## Fitur
* Manajemen skema formulir (JSON format)
* Validasi data dan pengerjaan formulir
* Manajemen batas waktu pengerjaan
* Pengolahan data analitik dan ekspor laporan

## Tech Stack
* ASP.NET Core
* SQL Server
